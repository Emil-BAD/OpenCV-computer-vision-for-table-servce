# Определение занятости столика

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Запустить скрипт:

```bash
python main.py --video video/video2.mp4
```

3. Дополнительные флаги:

```bash
python main.py --video video/video2.mp4 --output output.mp4
python main.py --video video/video2.mp4 --show
```

По умолчанию скрипт работает без окна предпросмотра и сохраняет результат в `output.mp4`.

## Какое видео и какой столик выбраны

- В текущей настройке проекта используется видео `video/video2.mp4`.
- Столик выбирается вручную на первом кадре с помощью `cv2.selectROI(...)` (Скриншот выбранного столика `choice.png`).
- После выбора зона столика расширяется на `padding = 80` пикселей с каждой стороны.

## Логика детекции событий

- Для поиска людей в кадре используется YOLO-модель `yolov8n.pt`.
- Человеком считается только объект с `class id = 0`.
- Столик считается занятым, если рамка человека пересекается с выбранной зоной столика.
- Столик не считается свободным сразу после пропуска детекции.
- Скрипт ждет `free_confirmation_sec = 4.0` секунды перед подтверждением, что столик освободился.
- Это уменьшает количество ложных событий, если человек временно пропал из распознавания.
- В журнал событий записываются два основных типа событий:
  - `became_occupied`
  - `became_free`
- После обработки видео скрипт рассчитывает интервалы между:
  - моментом освобождения столика
  - моментом подхода следующего человека
